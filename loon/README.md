# Loon 模板

本目录保存 Sub-Store / Loon 使用的公开配置模板。

## 文件说明

| 文件 | 用途 | 当前版本 |
|---|---|---|
| `loon.conf` | Loon 指定 Wi-Fi 自动直连模板 | `v2026.07.11-1` |

## Raw URL

```text
https://raw.githubusercontent.com/vhdapc/proxy-configs/main/loon/loon.conf
```

## 当前逻辑

- 指定 Wi-Fi：`狂乱贵公子`
- 指定 Wi-Fi 下策略组走 `DIRECT`
- 家中 DNS 指向：`192.168.6.1`
- 其他网络环境下按 Loon 自身规则分流

## 注意

该模板公开保存，因此不要加入真实节点、订阅链接、证书、密钥或 token。
